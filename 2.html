CSS

CSS stands for Cascading Style Sheets, and it's a computer language for specifying the presentation and styling of a document written in a markup language like HTML or XML. CSS is a fundamental technology of the World Wide Web, along with HTML and JavaScript. It allows developers to separate content from presentation, so that the visual layout of a web page can be established with style sheets, while the content can be put into HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
